GPT-SoVITS 支持多种音色克隆方式，质量和复杂度递增：

1. Zero-shot 克隆（当前实现）

- 数据需求: 3-10秒参考音频 + 对应文本
- 训练时间: 无需训练
- 质量: 基础，适合快速体验
- 用途: 快速原型、临时使用

2. Few-shot 微调

- 数据需求: 1-10分钟音频 + 对应文本（多条）
- 训练时间: 几分钟到1小时（需GPU）
- 质量: 中等，音色更稳定
- 产出: 自定义 GPT weights + SoVITS weights

3. 完整微调 (Full Fine-tuning)

- 数据需求: 30分钟-数小时音频
- 训练时间: 数小时（需较好GPU）
- 质量: 最佳，音色高度还原
- 用途: 专业级音色克隆

训练流程（Few-shot/Full）

音频数据 → 音频切分 → ASR标注 → 特征提取 → 训练 → 自定义权重

GPT-SoVITS 提供的工具：

- audio_slicer - 音频切分
- ASR - 自动标注
- GPT训练 - 训练语义模型
- SoVITS训练 - 训练声学模型
